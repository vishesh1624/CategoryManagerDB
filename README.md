# Fruits & Vegetables Category Command Center

A pure static portfolio project for category manager roles in quick commerce/e-commerce.

## How to use

Open `index.html` directly in your browser. No terminal commands are required.

## How to deploy on Vercel

1. Create a new GitHub repository.
2. Upload all files in this folder directly to the repository root.
3. Go to Vercel and import the GitHub repository.
4. Framework Preset: Other.
5. Build Command: leave blank.
6. Output Directory: leave blank or use `.`.
7. Deploy.

## Files

- `index.html` - main app
- `styles.css` - styling
- `script.js` - simulation logic
